Quick Start Guide

Add VigilX to your website in 3 simple steps:

1

Add the VigilX SDK

Include the SDK script in your HTML head:

HTML
<script src="https://cdn.vigilx.dev/sdk/v1.js" async></script>
2

Initialize VigilX

Call vigilx.init() with your site key:

JavaScript
vigilx.init({
  siteKey: 'vx_live_xxxxxxxxxxxx',
  endpoint: 'https://api.vigilx.dev',
  mode: 'passive',
  sensitivity: 0.7,
  onScore: (result) => {
    console.log('Human score:', result.score);
    console.log('Action:', result.action);
  }
});
3

Handle Results

Use the score callback to make decisions:

JavaScript
vigilx.onScore((result) => {
  if (result.action === 'allow') {
    submitForm(); // Human verified
  } else if (result.action === 'challenge') {
    showCaptcha(); // Suspicious
  } else {
    blockRequest(); // Bot detected
  }
});
